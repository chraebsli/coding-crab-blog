# Pre-Commit Checklist

## Posts

### Text

- [ ] The Grammar is ok
- [ ] All links work
- [ ] All images work
- [ ] Gets displayed right
- [ ] Images
  - [ ] are optimized
  - [ ] have an alt text

### Frontmatter

- [ ] Has a `title`
- [ ] Has a `description`
- [ ] (opt) Has a `permalink`
- [ ] Has `tags`
- [ ] Has `categories`
- [ ] Has a creation `date` in the form of `YYYY-MM-DD HH:MM +0100`
- [ ] (opt) Has a modified date (`last_modified_at`) in the form of `YYYY-MM-DD HH:MM +0100`
- [ ] Has a filename with date in the form of `YYYY-MM-DD-title-of-article.md`

### Metadata

- [ ] In `sitemap.xml`
- [ ] In `feed.xml`

### Multilingual

- [ ] Has a `lang` in the frontmatter
- [ ] Has a `lang-ref` in the frontmatter
- [ ] If translated post, shows language switcher

## Pages

### Text

- [ ] The Grammar is ok
- [ ] All links work
- [ ] All images work
- [ ] Gets displayed right

### Frontmatter

- [ ] Has a `title`
- [ ] Has a `description`
- [ ] Has a `permalink`
- [ ] Has a `layout`
- [ ] Has `author_profile`

### Metadata

- [ ] In `sitemap.xml`
- [ ] In `feed.xml`
